## ✨ Destaques e Funcionalidades

> O projeto foi construído sem o uso de frameworks, focando em extrair o máximo de performance com **Vanilla CSS e JS**, criando uma interface luxuosa e altamente interativa.

- 💎 **Design Glassmorphism & Neon Glow:** Paleta noturna com efeito de vidro fosco, bordas translúcidas e iluminação Rose Gold.
- 💬 **Gatilho de Conversão Inteligente:** Balão de WhatsApp programado via JavaScript com `setTimeout` e `sessionStorage` para atrair a cliente após 7 segundos de navegação sem ser invasivo.
- 🗺️ **Mapa Integrado em Dark Mode:** Iframe do Google Maps estilizado via CSS com filtros de inversão de cores (`invert` e `hue-rotate`) para não quebrar a identidade visual escura do site.
- 🎞️ **Animações Otimizadas:** Uso da API `IntersectionObserver` para revelar elementos (`fade-up`, `fade-right`) à medida que o usuário rola a página, garantindo zero travamentos.
- 📱 **100% Responsivo:** O layout se adapta perfeitamente em telas de qualquer tamanho, do desktop ultrawide ao mobile, utilizando CSS Grid e Flexbox.
- 🔗 **Open Graph e SEO:** Meta tags configuradas para gerar cards visuais de alta qualidade ao compartilhar o link no WhatsApp e redes sociais.
